# HomeTube Cookie Export Extension

Minimal Chromium unpacked extension for HomeTube.

## What it does

- Detects its presence from the HomeTube web page
- Exports cookies for the active site's primary domain
- Copies the exported Netscape cookies text to your clipboard

## Install

1. Either load this folder directly, or download the ZIP bundle from HomeTube and unzip it
2. Open `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the unzipped `hometube-cookie-export` folder

## Use

1. Open the site you want cookies for and make sure you are signed in
2. Click the extension icon
3. Click **Copy Active Site Cookies**
4. Go back to HomeTube
5. Paste the cookies text into the cookies manager and save
